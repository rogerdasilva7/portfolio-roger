Sobre Mim

Comecei a estudar programação em 2024. Atualmente, estou cursando Análise e Desenvolvimento de Sistemas (ADS) e focado em construir projetos práticos para aprimorar minhas habilidades. Tenho grande interesse por tecnologia e estou sempre em busca de novos aprendizados, com o objetivo de evoluir constantemente tanto no aspecto pessoal quanto profissional.


Tecnologias que utilizo:

ReactJS JavaScript HTML CSS TypeScript Git GitHub Vite Tailwind CSS

Expedix

Sistema desenvolvido para a empresa Laslife com foco no controle de expedição logística. Construída com React, TypeScript, Tailwind CSS e Firebase, ela permite que usuários se cadastrem e façam login para acessar uma tabela de registros de retiradas de mercadorias, contendo nome, telefone, cidade, data, hora e expedidor, podendo adicionar, pesquisar e excluir registros.
